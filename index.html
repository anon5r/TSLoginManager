TSLoginManager by anonymizer

This tools made for "Trickster" (MMO RPG). Anybody can easier log-in multiple accounts and manage them.

アカウント情報を登録することで、簡単にゲームを起動することができるランチャーソフトウェアです。 IDとパスワードを登録すると、タスクトレイに表示されたアイコンからアカウントを選択するだけでゲームランチャーまたはゲームを直接起動できます。 複数アカウントある場合には、ウェブページからいちいちログインして･･･という手間が省け、アカウントの切り替えを素早く行えます。 トリックスターラブ-0-公式サイト（http://trickster.jp）をはじめ、日本国内の提携ゲームポータル(ハンゲーム、アットゲームズ、Lievo)からのログインに対応しています。 アカウント情報はパスワードが暗号化されて保存されるため、アカウント情報ファイルだけ持ち歩けば、ネットカフェなどでキーロガーによるアカウントハックされる心配も減ります。 また、ゲームの異常終了時にタスクバーが表示されなくなるというゲームの潜在的なバグへの対応や、ゲーム中のフリーズを自動的に検出し、ゲームの強制終了を補助する機能も搭載しています。 付加機能をプラグインとして搭載。 また、上記により開発環境さえあれば、どなたでもプラグインを開発・追加・公開することができます。 By registering your account information, is launcher software that can easily start the game. You register a password and ID, and select the account from the icon that is displayed in the task tray. Then you can start the game for directly, or start the game launcher simply. If more than one account, you can quickly and simply switch the account. You do not need to login from a web page. This tools supports for Trickster Japanese Official (http://trickster.jp) to log in. And more some game portal alliance in Japan (Hangame, Games at, Lievo) . The account information is encrypted stored by your passwords. Moreover, it also includes some functions: * The ability to automatically detect freeze. * Some respond to potential bugs in the game * Supported: Taskbar does not displayed at the time of abnormal termination of the game. * To help to kill the game when it a freeze. Equipped with additional functions as a plug-in.
